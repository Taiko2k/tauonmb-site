Tauon Music Box has several advanced features for power users. This page provides links to detailed documentation on GitHub.

## Lyrics

Tauon supports displaying synchronized and unsynchronized lyrics. For detailed information on setting up and using lyrics, see the [Lyrics Guide](https://github.com/Taiko2k/TauonMusicBox/wiki/Lyrics) on GitHub.

## Broadcasting

You can broadcast your music to other devices on your network. For setup and usage information, see the [Outbound Broadcasting Guide](https://github.com/Taiko2k/TauonMusicBox/wiki/Outbound-Broadcasting) on GitHub.

## Transcoding

Tauon can transcode audio files to different formats. Learn how to set up and use transcoding in the [Transcoding Guide](https://github.com/Taiko2k/TauonMusicBox/wiki/Transcoding-audio-files) on GitHub.

## Network Playback

Stream music from network servers:
- **Jellyfin** - Connect to your Jellyfin server
- **Airsonic** - Stream from Airsonic-compatible servers

Configuration details are available in the main settings menu.

## Scrobbling

Tauon supports multiple scrobbling services:
- **Last.fm** - Track your listening history
- **ListenBrainz** - Open-source alternative to Last.fm
- **Maloja** - Self-hosted scrobbling server

Set up your scrobbling accounts in the settings menu.

## Further Reading

For more advanced features, troubleshooting, and FAQs, visit the [Tauon Music Box Wiki](https://github.com/Taiko2k/TauonMusicBox/wiki) on GitHub.
