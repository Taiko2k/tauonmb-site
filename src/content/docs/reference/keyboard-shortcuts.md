Here's a comprehensive list of keyboard shortcuts available in Tauon Music Box.

## Playback Controls

| Shortcut | Action |
|----------|--------|
| `Space` | Play/pause |
| `-` / `+` | Seek backward/forward |
| `Shift` + `Left` / `Right` | Previous/next track |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Left` / `Right` | Cycle playlists |
| `"` | Show playing track (locate in playlist) |

## Playback Modes

| Shortcut | Action |
|----------|--------|
| `,` | Toggle repeat mode |
| `.` | Toggle shuffle mode |

## Search & Find

| Shortcut | Action |
|----------|--------|
| `Ctrl` + `F` | Find in current playlist |
| Start typing | Activate global search |

## Views

| Shortcut | Action |
|----------|--------|
| `Tab` | Enter gallery view (keyboard control mode) |

## Themes

| Shortcut | Action |
|----------|--------|
| `F2` | Cycle through available themes |
| `F3` | Toggle "Auto Theme" mode |

## Customizing Key Bindings

You can edit key bindings by modifying the `input.txt` file in the user data folder.

To find this folder:
1. Go to **MENU > Settings...**
2. Press **"Open data folder"**
3. Edit the `input.txt` file

For a complete list of available shortcuts and advanced key binding options, see the [GitHub wiki](https://github.com/Taiko2k/TauonMusicBox/wiki/Keyboard-Shortcuts).
