## Customizing Key Bindings

You can edit key bindings by modifying the `input.txt` file in the user data folder.

To find this folder:
1. Go to **MENU > Settings...**
2. Press **"Open data folder"**
3. Edit the `input.txt` file

For a complete list of available shortcuts and advanced key binding options, see the [GitHub wiki](https://github.com/Taiko2k/TauonMusicBox/wiki/Keyboard-Shortcuts).
