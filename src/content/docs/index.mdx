Tauon is a desktop music player designed for those who love their digital music collection. It's built on the idea that a music player should be simple to use but deep in functionality. It combines a clean, modern interface with the power features you need to organize and enjoy your library.

## Features


<CardGrid>
  <Card title="Wide codec support" icon="star">
    Supports most common codecs, including MP3, FLAC, OGG, OPUS, AAC, WavePak, APE and TTA
  </Card>
  <Card title="CUE sheet support" icon="document">
    Support for CUE sheets with gapless playback
  </Card>
  <Card title="Milkdrop Visualisations" icon="star">
    Supports Winamp Milkdrop files (coming soon)
  </Card>
  <Card title="Smart Playlists" icon="list-format">
    An original smart playlist system with powerful generators
  </Card>
  <Card title="Network Playback" icon="rocket">
    Stream from Jellyfin or Airsonic servers
  </Card>
  <Card title="Desktop Integration" icon="setting">
    MPRIS2 support for seamless desktop integration
  </Card>
  <Card title="Context Search" icon="magnifier">
    Search Genius, Bandcamp and Rate Your Music
  </Card>
  <Card title="Archive Import" icon="add-document">
    One-click archive extract and import!
  </Card>
  <Card title="MOD Support" icon="information">
    Supports tracker files like MOD and retro game soundtracks like SPC
  </Card>
</CardGrid>
