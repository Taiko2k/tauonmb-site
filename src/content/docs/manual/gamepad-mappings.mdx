---
title: Gamepad Mappings
description: Gamepad controller mappings for Tauon Music Box
---

A gamepad controller can be used as a form of limited UI control.

| Button | Action |
| --- | --- |
| `A` | Play selected |
| `B` | Stop |
| `Y` | Pause/resume |
| `X` | Toggle gallery |
| `LB` | Previous track |
| `RB` | Next track |
| `LS U/D` | Fast scroll |
| `RS U/D` | Volume control |
| `RS L/R` | Seek |
| `DPAD U/D` | Select U/D |
| `DPAD L/R` | Cycle playlists |

With **LT** held:

| Button | Action |
| --- | --- |
| `A` | Show selected |
| `B` | Play previous track |
| `Y` | Play random track time |
| `X` | Play random track |
| `LB` | Toggle repeat |
| `RB` | Toggle shuffle |
